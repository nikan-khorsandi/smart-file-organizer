# 📂 Smart File Organizer

A clean and extensible Python application that automatically organizes files into categorized folders.

---

## ✨ Features

- 📂 Organize files by type
- 🖼️ Images
- 📄 Documents
- 🎵 Audio
- 🎬 Videos
- 📦 Archives
- 📁 Automatically create missing folders
- ⚡ Fast and lightweight
- 🧩 Easy to extend
